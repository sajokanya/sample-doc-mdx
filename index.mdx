# Welcome to Atoti

Atoti is a comprehensive Data Analytics platform that combines a best-in-class calculation engine, visualization suite, and real-time OLAP cube into a single, powerful solution. Built by [ActiveViam](https://www.activeviam.com/), Atoti includes a free Community Edition that makes advanced analytics accessible to everyone.

## What is Atoti?

Atoti transforms how you work with data by providing:

- **Real-time OLAP Cube**: Explore data across hundreds of dimensions with instant responsiveness
- **Advanced Calculation Engine**: Perform complex analytics on massive datasets
- **Interactive Visualizations**: Create compelling dashboards without coding
- **What-if Analysis**: Build multiple scenarios without duplicating data
- **Seamless Integration**: Connect to multiple data sources with continuous queries

## Key Capabilities

### Scale and Performance
- Analyze **1 billion rows** on your laptop
- Scale up to **10 terabytes** in the cloud
- Real-time data processing and updates
- Memory-optimized calculations

### Data Integration
- Connect to multiple data sources simultaneously
- Continuous queries for real-time updates
- Support for various data formats and databases
- Stream processing capabilities

### Analytics and Visualization
- Interactive widgets in Jupyter notebooks
- Web-based dashboarding application
- Excel integration for familiar workflows
- Multidimensional analysis across hundreds of dimensions

### Collaboration and Sharing
- Share results with peers and partners
- Export to multiple formats
- Web-based collaboration tools
- Enterprise-grade security features
