## Getting started

Before diving into the technical details, make sure you have:

1. **A Cirrus Sync account** - Sign up at [cirrus-sync.com](https://cirrus-sync.com) if you haven't already
2. **API credentials** - Generate your API key from Settings > Developer in your account dashboard
3. **Development environment** - Set up your preferred development tools and testing environment

## Quick start

Here's a simple example to get you started with the Cirrus Sync API:

```javascript
const CirrusSync = require('cirrus-sync-sdk');

const client = new CirrusSync({
  apiKey: 'your-api-key'
});

// List all syncs
const syncs = await client.syncs.list();
console.log(syncs);
```
